## Key Configuration Files

### hardhat.config.js
```javascript
require("@nomicfoundation/hardhat-toolbox");

module.exports = {
  solidity: "0.8.20",
  networks: {
    sepolia: {
      url: process.env.SEPOLIA_RPC_URL,
      accounts: [process.env.DEPLOYER_PRIVATE_KEY],
    },
    mainnet: {
      url: process.env.MAINNET_RPC_URL,
      accounts: [process.env.DEPLOYER_PRIVATE_KEY],
    },
  },
  etherscan: {
    apiKey: process.env.ETHERSCAN_API_KEY,
  },
};
```

### serverless.yml (key sections)
```yaml
service: nextgen-economy
frameworkVersion: "3"

provider:
  name: aws
  runtime: nodejs18.x
  region: us-east-1
  stage: ${opt:stage, 'dev'}
  environment:
    ETH_RPC_URL: ${ssm:/nextgen/${self:provider.stage}/eth-rpc-url}
    CONTRACTS_TABLE: ${self:service}-contracts-${self:provider.stage}
  iam:
    role:
      statements:
        - Effect: Allow
          Action:
            - dynamodb:GetItem
            - dynamodb:PutItem
            - dynamodb:Query
          Resource: "arn:aws:dynamodb:${self:provider.region}:*:table/${self:service}-*"

functions:
  getTokenBalance:
    handler: functions/api/getTokenBalance.handler
    events:
      - httpApi:
          path: /tokens/{address}/balance
          method: get
```

## Naming Conventions

- **Contracts**: PascalCase with `NGE` prefix (e.g., `NGEToken`, `NGEMarketplace`)
- **Test files**: Mirror contract path with `.test.js` suffix
- **Lambda functions**: camelCase (e.g., `getTokenBalance`, `processTransfer`)
- **DynamoDB tables**: `{service}-{tableName}-{stage}` pattern
- **Environment variables**: UPPER_SNAKE_CASE

How It Works

1
Sensor data is hashed — The raw sensor payload (temperature, humidity, GPS, etc.) is converted to a unique cryptographic hash using keccak256.
2
Hash is anchored on-chain — The hash is recorded on the Ethereum blockchain via the AnchoredDeviceRegistry smart contract, creating a permanent, tamper-proof record.
3
Anyone can verify — By pasting the hash (or raw data) here, we check the blockchain to confirm the data was recorded and has not been altered.
